# Walk-Dog

Aplikacija je namijenjena vlasnicima pasa. Cilj aplikacije je vlasncima olakšati brigu za svoje ljubimce i pružiti im šetnju i brigu koju im vlasnici ne mogu adekvatno pružiti zbog ubrzanog načina života.

## Aplikacija pruža: 
- mogućnost rezervacije termina i vremenskog intervala šetnje
- mogućnost odabura šetača
- unos podataka vezan psa
- pružanje dodatnih usluga (kupanje, nokti)
- korisnici imaju mogućnost ostavljanje komentara i ocjena obzirom na pruženu uslugu
